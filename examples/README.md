## Running Examples

```bash
# Run with interpreter
sigil run examples/simple.sigil

# Run with JIT (faster)
sigil jit examples/benchmark.sigil

# Type check only
sigil check examples/evidentiality.sigil

# Get AI IR
sigil dump-ir examples/ir_demo.sigil
```
